## 1. Мета роботи
Ознайомлення з базовими можливостями бібліотеки **Pandas** для аналізу табличних даних, освоєння методів фільтрації, групування та первинної візуалізації даних.

## 2. Опис набору даних
Для роботи було обрано датасет **BMW Global Sales Dataset**, який містить інформацію про продажі автомобілів BMW у різних країнах.
- **Кількість записів:** 1000
- **Кількість ознак:** 14 (після очищення залишено 8 основних)
- **Основні параметри:** Рік, Країна, Модель, Тип двигуна, Ціна (USD), Кількість проданих одиниць тощо.

## 3. Хід виконання роботи

### Етап 1: Підготовка та очищення
* Завантажено дані через `pd.read_csv`.
* Обрано 8 ключових колонок для аналізу.
* Виконано перевірку на пропуски (NaN) та видалення некоректних записів.
* Проведено форматування: створено булеву ознаку `is_premium` (ціна > 80k) та перетворено рік у формат `datetime`.

### Етап 2: Дослідження числових показників
Згідно з аналізом вибірки:
* **Кількість "дорогих" авто (>70,000$):** 562 записи.
* **Середні витрати на маркетинг для преміум-сегменту:** ~$276,100.
* **Лідер продажів серед дорогих моделей:** BMW 3 Series у Канаді (1242 одиниці).

### Етап 3: Категоріальний аналіз (USA та серія X)
Було проведено порівняння продажів у США та популярності моделей лінійки "X":
* Всього записів по США: 88.
* Моделі серії X у США: 43.
* Записи, що не стосуються ні США, ні серії X: 478.

### Етап 4: Групування та порівняння
Порівняння ринків Австралії та Франції:
| Країна | Кількість записів | Середня ціна ($) |
| :--- | :--- | :--- |
| Australia | 98 | 75,850.89 |
| France | 100 | 75,490.86 |

### Етап 5: Візуалізація
Побудовано стовпчикову діаграму загальних продажів по роках. Результат збережено у файл `sales_plot.png`.

## 5. Висновки 
Під час виконання лабораторної роботи було опановано інструментарій бібліотеки Pandas.
На прикладі даних BMW було реалізовано складні фільтри (AND/OR/NOT), проведено агрегацію даних за допомогою groupby та визначено статистичні показники ринку.
Створений графік демонструє динаміку продажів, що дозволяє робити висновки про успішність бренду в різні роки.
